Cybrics 2019

Network Sender

打开后可以看到是一个ESMTP的验证和发送邮件的过程

提示给了压缩包密码，所以接下来需要去寻找压缩包

使用nmap扫描一下端口，发现110端口打开，存在pop3服务

SMTP与pop3的区别:

POP是负责邮件程序和邮件服务器收信的通讯协定，SMTP则是负责邮件服务器与邮件服务器之间的寄信的通讯协定

ESMTP则是在SMTP的基础上加了验证身份的过程

接下来忘记截图了，大致过程是使用telnet连接ugm.cybrics.net的110端口，使用fawkes和Combin4t1onXXY的base64编码后的字符串通过验证，stat命令列举邮箱的邮件数量，发现只有一份邮件，并且其中有大附件，retr 1命令收取邮箱中的第一份邮件的内容，即拿到zip文件，最后使用crack0Weston88vertebra解密即可
